# BC-Scoreboard Premium

Version: 1.1.0

Created by BCGAMING


## Features

- Modern scoreboard UI
- QB-Core Support
- ESX Support
- OneSync Support
- Live player list
- Player ping display
- Job tracking
- Police / EMS / Fire counters
- Staff badges
- Player search
- Custom logo support
- Premium configuration system


## Requirements

- FiveM Server
- OneSync Enabled
- QB-Core or ESX


## Installation

1. Upload `bc-scoreboard` into your resources folder.

2. Add:

ensure bc-scoreboard

to your server.cfg.


3. Restart your server.


## Configuration

Edit:

config.lua


Change:

Config.Framework = "qb"

or:

Config.Framework = "esx"



## Staff Badges

Add your license:

Config.StaffPermissions


Example:

["license:YOUR_LICENSE"] = {

    name = "Owner",

    icon = "👑"

}
